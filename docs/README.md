# ByteBattle2 Documentation

This directory contains all project documentation as required by the
[ByteBattle2 Constitution](../.specify/memory/constitution.md) — Principle V.

## Structure

```
docs/
├── README.md           # This file
├── features/           # Feature documentation (one file per feature)
└── adr/                # Architecture Decision Records
```

## Guidelines

- Every new feature MUST have a markdown file in `features/`.
- Significant technical decisions MUST be recorded as ADRs in `adr/`.
- See the constitution for full documentation requirements.
